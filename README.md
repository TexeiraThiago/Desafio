
# Tech Android Test

This application should be able to CRUD over two entities: Workout and Exercise.

The relation between them is 1 to M.

| Workout          |       |    Exercise       |
|------------------|-------|-------------------|
|Name:Int          |       | Name:Int          |
|Description:String|       | Image:Url/Uri     |
|Date:TimeStamp    |       | observation:String|
